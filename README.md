# הקו הבוחן

פלטפורמה לניתוח התחבורה הציבורית בישראל. מסך פתיחה אחד מוביל לשלושה כלים:

- **קו פח** — איתור קווים ריקים ובזבזניים, מועמדים לביטול או צמצום.
- **הקו המוזהב** — איתור הקווים המצטיינים, הסטנדרט שאליו כדאי לשאוף.
- **קו באג** — איתור תקלות גאומטריות במסלולי הקווים (עיקופים ובליטות מיותרים).

האתר הוא אפליקציית צד‑לקוח (React) ללא שלב בנייה — `index.html` טוען את הקוד
ישירות, ו‑Babel מקמפל בדפדפן. כלי "קו באג" יושב כאפליקציה עצמאית תחת `kavbug/`.

## יצירה

הפלטפורמה הגתה, כוונה ונאצרה על‑ידי **שלמה הרטמן**: הרעיון, הגדרת הכלים,
החלטות המוצר והעיצוב, ובחירת מה לשמור ומה לשנות. המימוש בפועל (קוד, עיצוב
וטקסטים) נכתב בעזרת מודל בינה מלאכותית לפי ההנחיות והכיוון שלו.

## נתונים

נתוני התחבורה הציבורית (GTFS ונתונים פתוחים) הם של **משרד התחבורה** (ומקורות
ממשלתיים אחרים), אינם בבעלות בעל האתר, וכפופים לרישיונות ולתנאי השימוש של
בעליהם. ספריות צד‑שלישי הנטענות על‑ידי האתר (React, Leaflet, Babel, Tailwind)
נותרות תחת הרישיונות שלהן.
