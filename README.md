# הקו הבוחן

פלטפורמה לניתוח התחבורה הציבורית בישראל. מסך פתיחה אחד מוביל לשלושה כלים:

- **קו פח** — איתור קווים ריקים ובזבזניים, מועמדים לביטול או צמצום.
- **הקו המוזהב** — איתור הקווים המצטיינים, הסטנדרט שאליו כדאי לשאוף.
- **קו באג** — איתור תקלות גאומטריות במסלולי הקווים (עיקופים ובליטות מיותרים).

האתר הוא אפליקציית צד‑לקוח (React) ללא שלב בנייה — `index.html` טוען את הקוד
ישירות, ו‑Babel מקמפל בדפדפן. כלי "קו באג" יושב כאפליקציה עצמאית תחת `kavbug/`.

## יצירה, כיוון ובעלות

הפלטפורמה הגתה, כוונה ונאצרה על‑ידי **שלמה הרטמן**: הרעיון, הגדרת הכלים,
החלטות המוצר והעיצוב, ובחירת מה לשמור ומה לשנות — כולם שלו. (הנתונים עצמם
אינם שלו — ראו להלן.)

המימוש בפועל (קוד, עיצוב וטקסטים) נכתב **בעזרת Claude** (מודל בינה מלאכותית של
Anthropic), לפי ההנחיות והכיוון של שלמה הרטמן. כלומר זוהי יצירה בליווי AI תחת
כיוון אנושי, ולא תוצר אוטומטי ללא בקרה.

## זכויות ונתונים

זכויות היוצרים על האתר — קוד המקור, העיצוב והטקסטים המקוריים — שמורות.
ראו פירוט מלא בקובץ [`LICENSE`](./LICENSE).

נתוני התחבורה הציבורית (GTFS ונתונים פתוחים) הם של **משרד התחבורה** (ומקורות
ממשלתיים אחרים), **אינם** חלק מהיצירה הזו ואינם בבעלות בעל האתר — הם כפופים
לרישיונות ולתנאי השימוש של בעליהם. כך גם ספריות צד‑שלישי הנטענות על‑ידי האתר
(React, Leaflet, Babel, Tailwind), שנותרות תחת הרישיונות שלהן.
